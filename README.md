# Matias Alario Portfolio

Portfolio personal construido con Vite, React y TypeScript.

## Scripts

- `npm run dev`: servidor local de desarrollo.
- `npm run build`: validacion TypeScript y build de produccion.
- `npm run preview`: vista local del build.

## Deploy

El proyecto esta preparado para Vercel y GitHub Pages. En GitHub Pages, el workflow en
`.github/workflows/deploy.yml` compila la app y publica el directorio `dist`.
